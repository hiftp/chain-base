E-Commerce HitPay Payment Gateway Payment Provider

HitPay Payment Gateway Plugin allows merchants to accept PayNow QR, Cards, Apple Pay, Google Pay, WeChatPay, AliPay and GrabPay Payments

Configuration

  • Navigate to the Website and click on the Configuration menu.
  • Now click on the submenu of Payment Providers and open HitPay Payment Gateway Payment method.
  • Enter credentials for HitPay Payment Gateway Payment method.

Checkout

  • Select HitPay Payment Gateway Payment method and click on the Pay Now button.
  • If create payment is successful, it would redirect to the HitPay Payment Gateway interface.

HitPay Payment Gateway Interface

  • You can select any payment method and click on Pay Now button.
  • If payment is successful, it would redirect to the website automatically.
  • If payment is failed or error, still it would redirect to the website automatically but order payment status will be updated via webhook.

Order Confirmation

Payment Transaction

  • Navigate to the Website and click on the Configuration menu.
  • Now click on the submenu of Payment Transactions and check our transaction.

Refund

  • In the payment transaction, click the payment link eg: PBNK1/2023/00004.
  • You see the below screen and click the 'Refund' button and enter the amount to refund.
